# Flexbox

**Flexbox**（CSS Flexible Box Layout Module）は、アイテムを行または列に並べるための 1 次元のレイアウト方法です。

## Flexbox の主要な特徴

### 1. 一次元レイアウト

- 要素を一方向に配置（行または列）
- 要素間の配置や配分を柔軟に制御
- コンテナ内の要素を HTML の順序に関係なく配置

### 2. レスポンシブ対応

- 画面サイズに応じて要素が自動的に伸縮
- 異なる画面サイズでも一貫したレイアウト

### 3. 簡単な中央揃え

- 垂直方向の中央揃えが簡単に実現
- 複数の軸での配置制御が直感的

以下のようなレイアウトを簡単に実現できます。

<FlexContainer style={{ margin: '20px' }}>
    <FlexItem>Item 1</FlexItem>
    <FlexItem>Item 2</FlexItem>
    <FlexItem>Item 3</FlexItem>
</FlexContainer>

:::tip

初期の頃にCSSを学んだ人は float プロパティを段組みに使っていたかもしれませんが、Flexboxではそのほとんどを置き換えることができます。
floatでしか作れないレイアウトは画像の回り込みなどの特殊なケースのみです。忘れましょう。
:::
