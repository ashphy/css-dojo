# 演習課題

Flexboxを使って実用的なレイアウトを段階的に作成してみましょう。各レベルをクリアして、Flexboxマスターを目指しましょう！

## レベル1: 基本的な配置

まずは基本的な配置から始めましょう。

### 課題1-1: 水平中央配置

3つの要素を横一列に並べ、コンテナの中央に配置してください。

**期待される結果:**
- 3つの要素が水平方向に並んでいる
- 要素全体がコンテナの中央に配置されている
- 要素間に適度な余白がある

<Playground
  htmlCode={`<div class="container">
  <div class="item">アイテム1</div>
  <div class="item">アイテム2</div>
  <div class="item">アイテム3</div>
</div>`}
  cssCode={`.container {
  height: 200px;
  background-color: #f0f0f0;
  border: 2px solid #ccc;
  /* ここにFlexboxのプロパティを追加してください */
}

.item {
  background-color: #007bff;
  color: white;
  padding: 15px 20px;
  border-radius: 5px;
}`}
/>

**ヒント:** `display: flex`, `justify-content`, `gap` プロパティを使用してください。

<details>
<summary>解答例を見る</summary>

```css
.container {
  height: 200px;
  background-color: #f0f0f0;
  border: 2px solid #ccc;
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 20px;
}
```

</details>

## レベル2: 実用的なコンポーネント

### 課題2-1: ナビゲーションバー

ロゴを左側に、メニューを右側に配置したナビゲーションバーを作成してください。

## レベル3: 複合レイアウト

より複雑なレイアウト構造を作成してみましょう。

### 課題3-1: ヘッダー・サイドバー・メインレイアウト

Webサイトの基本的な構造を作成してください。ヘッダー、サイドバー、メインコンテンツの3つのセクションからなります。

<Playground
  htmlCode={`<div class="layout">
  <header class="header">
    <h1>Webサイトのタイトル</h1>
  </header>
  <div class="main-layout">
    <aside class="sidebar">
      <nav>
        <ul>
          <li><a href="#">メニュー1</a></li>
          <li><a href="#">メニュー2</a></li>
          <li><a href="#">メニュー3</a></li>
        </ul>
      </nav>
    </aside>
    <main class="content">
      <h2>メインコンテンツ</h2>
      <p>ここにメインのコンテンツが入ります。サイドバーは固定幅で、メインコンテンツは残りのスペースを占有します。</p>
    </main>
  </div>
</div>`}
  cssCode={`.layout {
  height: 400px;
  border: 2px solid #ccc;
}

.header {
  background-color: #333;
  color: white;
  padding: 20px;
  text-align: center;
}

.header h1 {
  margin: 0;
  font-size: 24px;
}

.main-layout {
  /* ここにFlexboxのプロパティを追加してください */
  height: calc(100% - 80px);
}

.sidebar {
  background-color: #f8f9fa;
  padding: 20px;
  border-right: 1px solid #ddd;
  /* ここにflexプロパティを追加してください */
}

.sidebar ul {
  list-style: none;
  margin: 0;
  padding: 0;
}

.sidebar li {
  margin-bottom: 10px;
}

.sidebar a {
  color: #333;
  text-decoration: none;
}

.sidebar a:hover {
  text-decoration: underline;
}

.content {
  padding: 20px;
  /* ここにflexプロパティを追加してください */
}

.content h2 {
  margin-top: 0;
}`}
/>

**期待される結果:**
- ヘッダーが上部に固定されている
- サイドバーが固定幅（約200px）で左側に配置されている
- メインコンテンツが残りのスペースを占有している

<details>
<summary>解答例を見る</summary>

```css
.main-layout {
  display: flex;
  height: calc(100% - 80px);
}

.sidebar {
  background-color: #f8f9fa;
  padding: 20px;
  border-right: 1px solid #ddd;
  flex: 0 0 200px;
}

.content {
  padding: 20px;
  flex: 1;
}
```

</details>

## レベル4: 高度なレイアウト

### 課題4-1: フッター固定レイアウト

コンテンツの量に関係なく、フッターを常にページの下部に固定するレイアウトを作成してください。

<Playground
  htmlCode={`<div class="page">
  <header class="page-header">
    <h1>ヘッダー</h1>
  </header>
  <main class="page-main">
    <h2>メインコンテンツ</h2>
    <p>コンテンツの量が少なくても、フッターはページの下部に固定されます。</p>
    <p>コンテンツを追加してみてください。</p>
  </main>
  <footer class="page-footer">
    <p>&copy; 2024 My Website. All rights reserved.</p>
  </footer>
</div>`}
  cssCode={`.page {
  height: 400px;
  border: 2px solid #ccc;
  /* ここにFlexboxのプロパティを追加してください */
}

.page-header {
  background-color: #007bff;
  color: white;
  padding: 20px;
  text-align: center;
}

.page-header h1 {
  margin: 0;
}

.page-main {
  padding: 20px;
  background-color: white;
  /* ここにflexプロパティを追加してください */
}

.page-footer {
  background-color: #333;
  color: white;
  padding: 15px;
  text-align: center;
}

/>

**期待される結果:**
- ヘッダーが上部に配置されている

<details>
<summary>解答例を見る</summary>

```css
}
```

</details>
