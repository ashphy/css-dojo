# Flexboxの基礎

# フレックスコンテナ

フレックスコンテナは、`display: flex` または `display: inline-flex` を設定した要素です。
この要素が Flexbox レイアウトの親要素となります。
フレックスコンテナの直接の子要素は、フレックスアイテムと呼ばれます。


```html
<div class="flex-container">
  <div>フレックスアイテム 1</div>
  <div>フレックスアイテム 2</div>
  <div>フレックスアイテム 3</div>
</div>
```

```css title="CSS"
.flex-container {
  display: flex;
}
```

<FlexContainer>
    <FlexItem>フレックスアイテム 1</FlexItem>
    <FlexItem>フレックスアイテム 2</FlexItem>
    <FlexItem>フレックスアイテム 3</FlexItem>
</FlexContainer>
